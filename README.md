# Sensu

Aplicación móvil de monitoreo y cuidado personal desarrollada con React Native y Expo.

## Características

- Gestión de dispositivos de seguimiento (Eview)
- Alertas y notificaciones en tiempo real
- Ubicación y geofencing con Google Maps
- Perfil médico: condiciones, medicamentos y contactos de emergencia
- Servicio de conserjería
- Botón SOS

## Requisitos

- Node.js 18+
- Expo CLI

## Instalación

```bash
npm install
cp .env.example .env  # Configurar variables de entorno
npx expo start
```
